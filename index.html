Beaming

Beaming is a free, open source puzzle game that involves directing beams through a hexagonal grid. The rules and solution for each puzzle must be inferred through interaction and visual cues.

Help

If you have a suggestion or you've found a bug, you can report it on the issue tracker. Note that the current state of any given puzzle is stored in the URL (/[puzzle-id]/[state]) and should be included when requesting help. You can of course also use this URL to share a specific state of a puzzle with others. If you want to avoid spoilers, remove the state from any URL before you visit it.

Thanks!

This game was primarily inspired by The Talos Principal and The Witness, two highly recommended puzzle games. Additionally, this game would not have been possible without Paper.js.
